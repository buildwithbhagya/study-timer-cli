# Study Timer

A Python-based CLI study timer that helps manage focused study sessions with break intervals using loop control and user input.

## Features
- Takes user input for study and break duration
- Uses loops to automate study-break cycles
- Simple command-line interface

## How to Run
1. Make sure Python is installed
2. Run the file using:
   python filename.py
